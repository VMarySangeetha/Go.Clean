# 🌍 GO.Clean

GO.Clean is a smart municipal waste management and citizen engagement platform designed to improve city cleanliness through technology. Citizens can report waste-related issues, earn rewards, participate in community events, and help build a cleaner environment.

## 🚀 Live Demo

- Frontend: https://go-clean01.netlify.app/
- Backend API: https://go-clean-8c5n.onrender.com/

---

## 📌 Features

### 👤 Authentication
- Email & Password Login
- Google Sign-In using Firebase Authentication
- Secure user session management

### 🗑️ Waste Issue Reporting
- Report garbage dumping, overflowing bins, damaged bins, etc.
- Upload issue photos
- GPS / location support
- Track submitted reports

### 📊 User Dashboard
- View submitted reports
- Track report status:
  - Pending
  - In Progress
  - Completed
- Earn coins for completed reports

### 🎁 Rewards System
- Redeem earned coins
- Download digital certificates
- Citizen engagement gamification

### 🏙️ Municipal Events
- View upcoming municipal / community cleanliness events
- Responsive event cards with schedule details

### 👨‍💼 Admin Panel
- Role-based access
- Manage reports
- Update status
- User monitoring

### 🌐 Multi-language Support
Supports 10+ languages for accessibility and wider reach.
